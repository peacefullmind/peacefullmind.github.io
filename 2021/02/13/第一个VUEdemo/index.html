安装Node.js

从nodejs官网下载安装，验证：命令行中，node-v出现版本号，说明成功。

安装npm的国内镜像

node环境中包括，npm包管理器，由于国外，网速问题，我们安装国内镜像
命令行中，
npm install -g cnpm -registry=http://registry.npm.taobao.org
然后可以用cnpm来代替cpm安装依赖包

安装vue-cli脚手架构建工具

命令行
cnpm install -g vue-cli

用vue-cli构建项目

选择一个目录放项目，命令行转到该目录下

意思是初始化一个项目，利用的是webpack打包和压缩。
然后按照提示输入选项
